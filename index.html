    • TrayRSS

    The goal of this tool is to offer the user a fast way to watch news given by an RSS or ATOM feed by tray notifications. To read the news, it will open a browser. For more information see http://trayrss.nullpointer.at

    goal

    The goal of this tool is to offer the user a fast and cheap way to read news given by an RSS or ATOM feed by tray notifications. To read the news, it will open a browser. features

    • read RSS Feeds
    • Tool Tip Notification
    • simple
    • platform independet
